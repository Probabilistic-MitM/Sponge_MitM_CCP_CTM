# Verification Experiments

This folder contains code for verification experiment of the meet-in-the-middle (MitM) process, including trail search, implementation, and visualization.

## Files

- **`Experiment_search.py`**  
  Searches for trails/paths used in the experiment.

- **`Experiment_verify.py`**  
  Verifies the MitM process in an experiment setting.  
  The results are written to **`result.txt`**.

- **`Experiment_draw.py`**  
  Draws/visualizes the experimental trail/path.
